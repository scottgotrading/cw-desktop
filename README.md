# ChartsWatcher Desktop Edition

A desktop app for **ChartsWatcher**, built by SCOTTGO. It wraps the
ChartsWatcher web app in a native Windows shell and adds:

- A custom title bar with a market clock, timezone picker, and update indicator
- Broker-style themes (DAS, IBKR, Webull, ThinkOrSwim, Lightspeed, Sterling
  and more), with a full theme editor
- A multi-monitor window launcher for your CW screens and your own trading apps
- Always-on-top popout windows
- Optional encrypted login autofill

You need your own ChartsWatcher account to use it. The app is licensed: on first
run you activate it with a licence key, which you can request in-app and have
emailed to you. One key works on up to two of your own devices.

---

## Download

**[Download the latest version from the Releases page](../../releases/latest)**

1. Download the `.exe` installer from the latest release.
2. Run it and follow the prompts.
3. On first run, agree to the terms and activate your licence. No key yet? Choose
   **Email my key**, enter your email, then paste the key back in to activate.
4. The Install Guide PDF is attached to the release if you want a walkthrough.

### "Windows protected your PC" warning
The installer is new and not yet widely downloaded, so Windows SmartScreen may
show a blue warning. This is expected for a new app and clears as more people
install it.

To continue: click **More info**, then **Run anyway**.
